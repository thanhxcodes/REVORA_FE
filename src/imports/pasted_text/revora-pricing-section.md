Design a premium pricing and membership section for a Gen Z fashion resale platform called “REVORA”, inspired by the red luxury aesthetic of the FASHIONORA website.

The pricing system must feel realistic, modern, and marketplace-oriented, similar to boosting systems on Facebook Marketplace or Chợ Tốt, but redesigned with a premium Gen Z fashion-tech experience.

Overall Style:
- Dark crimson red + burgundy luxury theme
- Fashion-forward UI
- Elegant glassmorphism cards
- Soft shadows and glowing hover effects
- Rounded corners
- Modern typography
- TikTok + luxury fashion marketplace vibe
- Clean but visually premium

The pricing section must contain ONLY TWO MAIN PACKAGE TYPES:

1. Posting Packages (Gói Đăng Bài)
2. Featured Packages (Gói Nổi Bật)

Each package type contains:
- Day package
- Week package
- Month package

The longer duration should provide:
- more posting credits
- cheaper price per credit
- better overall value

Package logic:
- Packages do NOT directly unlock features permanently
- They add temporary credits/usages to the user account
- Credits have expiration dates
- When credits expire, unused credits are removed automatically

Example:
“10 posting credits valid for 7 days”
